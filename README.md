# 2026全球签证免签政策地图

[![示意图](示意图.jpg)](示意图.jpg)

这是一个基于中国护照最新签证政策开发的交互式地图项目。

## 🌟 功能特点

- **实时更新**：涵盖免签、落地签、电子签及申根/美签免签政策
- **交互体验**：支持缩放、悬停查看详情
- **导出功能**：支持将当前地图导出为 PNG 或 PDF
- **多地图集成**：在一个页面中集成7种不同类型的签证政策地图
- **📊 访问统计**：实时追踪访客地理分布，可视化展示访问数据（新功能！）

## 🚀 快速开始

### 查看签证地图
1. 下载项目文件
2. 在浏览器中打开 `index.html`
3. 点击顶部标签页切换不同的签证政策地图
4. 鼠标悬停在国家上查看详细信息

### 启用访问统计（可选）
如果您想追踪网站访问情况并在地图上展示：
1. 查看 `快速部署指南.md` 了解三步配置流程
2. 配置 Firebase（免费，约 10 分钟）
3. 访问 `访问统计地图.html` 查看统计数据

详细说明请参考：
- 📘 [`快速部署指南.md`](快速部署指南.md) - 三步完成配置
- 📗 [`访问统计使用指南.md`](访问统计使用指南.md) - 详细功能说明
- 📕 [`FIREBASE_SETUP.md`](FIREBASE_SETUP.md) - Firebase 配置教程

## 📊 支持的地图类型

### 签证政策地图
1. **全球汇总** - 所有签证政策的综合展示
2. **中国护照** - 中国公民的免签/落地签/电子签政策
3. **美国签证** - 持有美国签证的可免签国家
4. **英国签证** - 持有英国签证的可免签国家
5. **日本签证** - 持有日本签证的可免签国家
6. **澳洲签证** - 持有澳大利亚签证的可免签国家
7. **申根签证** - 申根区及相关免签政策

### 访问统计地图（新增）
- **访客分布热力图** - 在中国地图上显示访客城市分布
- **访问统计面板** - 总访问量、城市数、国家数、今日访问
- **城市排行榜** - 显示访问量最多的前 20 个城市
- **实时更新** - 每 30 秒自动刷新数据
