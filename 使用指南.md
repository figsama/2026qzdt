# 全球签证免签政策地图使用指南

## 项目概述

这是一个综合性的全球签证免签政策可视化项目，提供交互式地图，帮助用户了解不同国家的签证政策。通过直观的颜色编码和详细的鼠标悬停信息，用户可以快速掌握：

- 各国对中国护照的签证政策
- 持有美签/申根签/英签/日签/澳签可以免签的国家
- 过境免签政策
- 签证签发国的分布

## 文件结构

### 主要文件

1. **`签证地图总览.html`** - 集成地图总览页面
   - 集成了7种不同类型的签证政策地图
   - 支持一键切换查看不同类型的签证政策
   - 包括：全球汇总、中国护照、美国签证、英国签证、日本签证、澳洲签证、申根签证
   - 适合对比不同签证政策和综合了解

### 数据文件

2. **`全球签证免签政策汇总.md`** - 数据汇总文档
   - 以文本形式汇总所有签证政策信息
   - 包含详细的国家列表和政策说明
   - 适合数据分析和备份

3. **`使用指南.md`** - 本文档
    - 详细的使用说明和功能介绍

## 地图功能说明

### 颜色编码系统

#### 全球汇总地图颜色说明
- **签证签发国（蓝色 #1e40af）** - 美签、英签、日签、澳签、申根签发国
- **有条件免签（橙色 #f59e0b）** - 需要特定条件（如持有有效签证）的免签
- **过境免签（蓝色 #3b82f6）** - 允许一定时间内过境免签
- **中国护照免签（绿色 #10b981）** - 中国公民免签国家
- **中国护照落地签（浅绿 #34d399）** - 中国公民落地签国家
- **中国护照电子签（青绿 #a7f3d0）** - 中国公民电子签国家

#### 单一政策地图颜色说明
- **签发国（蓝色 #1e40af）** - 相应签证的签发国家
- **免签国家（绿色 #059669）** - 可免签入境的国家
- **落地签（浅绿 #34D399）** - 可落地签入境的国家
- **电子签（青绿 #A7F3D0）** - 可电子签入境的国家
- **过境免签（蓝色 #3b82f6）** - 允许过境免签的国家

## 使用方法

### 1. 打开地图总览

1. 直接在浏览器中打开 `签证地图总览.html` 文件
2. 页面顶部显示7个标签页按钮，每个对应不同类型的签证政策地图

### 2. 选择合适的地图类型

根据您的需求点击相应的标签页：

- **全球汇总** → 展示所有签证政策汇总信息，适合快速了解全球签证格局
- **中国护照** → 专门展示中国护照持有者的签证政策（免签/落地签/电子签）
- **美国签证** → 展示持有美国签证可以免签的国家，适合有美签的用户
- **英国签证** → 展示持有英国签证可以免签的国家，适合有英签的用户
- **日本签证** → 展示持有多次日本签证可以免签的国家，适合有日签的用户
- **澳洲签证** → 展示持有澳大利亚签证可以免签的国家，适合有澳签的用户
- **申根签证** → 展示申根区国家及持有申根签可免签的国家，适合欧洲旅行规划

### 2. 查看政策信息

1. 打开相应的HTML文件（直接在浏览器中打开）
2. 将鼠标悬停在感兴趣的国家上
3. 阅读弹出的政策信息
4. 根据颜色编码快速识别政策类型


### 数据更新

- 数据基于2026年最新签证政策
- 如需更新数据，请修改 `签证地图总览.html` 文件中的相应 `countryData` 或 `areas` 数组
- 建议定期检查官方签证政策更新
- 数据同时保存在 `全球签证免签政策汇总.md` 中以供参考
